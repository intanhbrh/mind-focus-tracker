Elevare — Automatic Number Plate Recognition Pickup App

Elevare is an ANPR-powered school dismissal system that replaces manual name-calling with automated parent detection. When a camera reads a car's plate at the school entrance, it triggers a real-time push notification to the student's device to proceed to the pickup zone. The system includes a mobile app for students, a backend server handling plate-to-student matching, push notifications via Expo, OTP-based login using school emails, and a live foyer display board showing student names by lane.

Challenges Faced

Joining an already-developed codebase was initially difficult — understanding the existing structure, conventions and logic took focused effort in Week 1. In Week 2, SMTP and email server restrictions blocked OTP implementation, requiring exploration of multiple solutions including SendGrid domain authentication.

💡

Lessons Learned

Reading existing code before making changes saves significant debugging time. When one solution fails (personal SMTP), pivoting systematically to alternatives (SendGrid, school email server) is more productive than persisting with a blocked approach. Documentation and asking targeted questions to supervisors is key.

🎯
